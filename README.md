## 概述
本项目是在指导手册引导下，从零开始构建一个基于RISC-V架构的简单操作系统。

## 环境配置
- **操作系统**: Ubuntu 22.04.5 LTS
- **工具链 (GCC)**: riscv64-unknown-elf-gcc 12.2.0
- **模拟器 (QEMU)**: 8.2.0

## 编译运行
- 在实验文件夹根目录下，执行以下命令进行编译： `make` 该命令会编译所有内核源文件，并链接生成最终的可执行文件 `kernel.elf`。 
- 编译成功后，执行以下命令来启动QEMU并运行内核： `make qemu`（直接执行`make qemu`也可以）
- 如果需要清理所有编译生成的文件（如.o, .elf文件），可以执行： `make clean`

## 项目结构
 - 由于实际上实验6是最后一个完成的实验，所以实验6文件夹里的是整个实验的最终版本（包括了实验7和扩展项目的内容）。除此以外，每个实验文件夹都包含了到对应实验为止的内容，均可独立编译运行，输出对应实验的测试内容。
- 每个实验文件夹内部都包含一个md文件，写有更详细的说明；每个实验文件夹内部是类似的结构，主要内容在`./kernel`文件夹下随实验进行而累积

### 子文件夹说明
- **include:** 包含头文件
- **scripts:** 包含脚本文件，例如`kernel.ld`
- **tools:** 包含`mkfs.c`，构建文件系统镜像
- **user:** 包含用户态文件
- **kernel:** 包含核心实验代码
  - `boot`包含系统启动引导代码
  - `mm`包含系统内存相关代码
  - `proc`包含系统进程相关代码
  - `trap`包含系统中断相关代码
  - `fs`包含系统文件相关代码
